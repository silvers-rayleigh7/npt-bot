# Приложение НПТ (офлайн-гид)

Статус: 📋 план, кода ещё нет.

## Зачем
На объекте интернет, скорее всего, отключат. Telegram-бот без интернета не работает, поэтому
офлайн-режим на тропе реализует именно приложение.

## Идея
- **PWA или нативное** приложение, которое заранее (пока есть интернет) скачивает пакет тропы:
  сюжеты, аудио, карту.
- **GPS работает офлайн** (спутники) → приложение определяет точку и проигрывает локальный сюжет.
- **On-device TTS** для голоса без интернета: Silero / Piper (бесплатные, русский, работают офлайн).
- Контент берётся из общего слоя сюжетов (см. `tg-bot/content/`, позже вынесем в `shared/`).

## Открытые вопросы
- PWA vs нативное (iOS/Android).
- Формат пакета тропы (предсгенерированные mp3 + JSON карта точек).
- Синхронизация контента с ботом/сайтом.
