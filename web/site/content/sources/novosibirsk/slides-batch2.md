## Слайды 6+7 — Обратные и некорректные задачи (объединить в 1 сюжет)
**Лаврентьев-младший** (Михаил Михайлович, 1932–2010, сын М.А. Лаврентьева) — один из основателей теории
некорректных задач, создатель новосибирской школы обратных задач.
- Прямая задача: по причине → следствие, $y = A(x)$. Обратная: по следствию восстановить причину (взрыв
  как источник информации → сейсморазведка).
- **Корректность по Адамару** = существование + единственность + устойчивость. Обратные задачи ломают
  устойчивость. Пример — **задача Коши для уравнения Лапласа**: $u(x,0)=0$, $u_y(x,0)=\tfrac{1}{n}\sin(nx)$
  ⇒ $u=\tfrac{1}{n^2}\sin(nx)\,\mathrm{sh}(ny)$; при $n\to\infty$ данные исчезают, а решение растёт без предела.
- **Регуляризация Лаврентьева**: обратить $Ax=y$ напрямую неустойчиво ($A^{-1}$ раздувает шум). Для $A=A^*\ge0$
  заменяют устойчивым уравнением второго рода: $(A+\alpha I)x = y$. Добавка $\alpha I$ держит обусловленность
  конечной. Тот же принцип — L2-регуляризация Тихонова; та же форма $(K+\lambda I)^{-1}$ в kernel-методах ML
  (возникла у Лаврентьева задолго до них).
- **Условная корректность** (фундамент школы): если сузить решение до компактного множества, из единственности
  следует устойчивость.
- Школа: В.Г. Романов (обратные задачи для гиперболических/кинетических уравнений; восстановить скорость в
  $u_{tt}=c^2(x)\Delta u$ по граничному отклику = сейсморазведка). С.И. Кабанихин (численные методы). А.Л. Бухгейм
  (метод Бухгейма–Клибанова 1981, оценки Карлемана, глобальная единственность многомерных задач).
- Применения: УЗИ/КТ, сейсморазведка/геофизика, астрономия, дистанционное зондирование. Конференция
  «Теория и численные методы решения обратных и некорректных задач» (НГУ, ИВМиМГ, Институт математики им. Соболева).

## Слайд 11 — Дендрохронология  [регион под вопросом: природная тема]
Каждый год дерево откладывает новый слой древесины: благоприятный год → широкое кольцо, засушливый/холодный →
узкое. Последовательность широких и узких колец = уникальный «штрихкод времени». **Перекрёстная датировка**:
у разных деревьев, росших рядом, последовательность колец одинакова; совмещая рисунки, «сшивают» древесную
летопись из нескольких поколений деревьев (на сотни лет назад). Дендрохронология датирует постройки,
реконструирует климат, историю лесов. Интерактив: посетитель получает два спила и ищет совпадающий рисунок.

## Слайд 12 — Скрытая математика в природе (масштаб)  [природная тема]
Живой организм нельзя просто увеличить. Кислород поступает через поверхность (альвеолы, капилляры), а питать
надо весь объём. Размерный анализ: площадь газообмена $S_л \sim L^2$, масса $M \sim L^3$, обмен $B \sim S_л$.
Отсюда $B \sim L^2 \sim M^{2/3}$. Но эксперимент (мышь→слон) даёт **закон Клайбера** $B \sim M^{3/4}$ — обмен
растёт быстрее, чем 2/3. «Лишнее» — в **фрактальном** устройстве доставки (сосуды, бронхи). Фрактал: размерность
дробная, $N(r)\sim r^{-D}$; сеть сосудов/бронхов ветвится и почти заполняет объём, измеренная размерность ≈2,7–2,9
(между плоскостью 2 и объёмом 3) → «почти-объёмность» и даёт показатель 3/4.

## Слайд 13 — Почему старый лес нельзя посадить заново  [природная тема]
Лес — не только деревья: почва, корни, грибы, мхи, насекомые, птицы. За десятилетия копятся связи между видами →
устойчивая экосистема. Пока лес растёт, он связывает CO₂ и переводит в древесину/корни/почву. До половины
углерода — в почве и подстилке, не в стволах. Старовозрастный тёмнохвойный лес = результат многовековой
**сукцессии** (смена сообществ после нарушений: пожар, вырубка, старение). Фотосинтез:
$6CO_2 + 6H_2O \to C_6H_{12}O_6 + 6O_2$. Часть углерода остаётся надолго. Лес ценен накопленными запасами,
процессами и связями, которые формируются десятилетиями — потому быстро «посадить заново» нельзя.
