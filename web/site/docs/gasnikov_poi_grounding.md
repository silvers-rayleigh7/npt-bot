# Gasnikov lectures → POI grounding analysis (v2)

**Дата**: 2026-05-28

**Маршруты**: Иннополис (8 POI) + Сколково (7 POI) = 15 точек

Критерий: для каждой лекции спрашиваем — есть ли на реальном маршруте конкретное место, где этот сюжет осмысленно работает, и **остановится ли критический посетитель послушать**.

## Перечень существующих POI (контекст)

**Иннополис:**
1. `university_entrance` — GPS/трилатерация, синхронизация часов

3. `robot_path` — SLAM, лидар, роботы-доставки
4. `tesla_marker` — индукция Фарадея, переменный ток
5. `boulevard_game` — равновесие Нэша–Вардропа, **парадокс Браесса**, дизайн правил
6. `sports_scaling` — ходьба/бег, шагомеры, масштаб тела
7. `forest_water_column` — высокие деревья, когезия, кавитация
8. `data_center_view` — дата-центры, тепло, физика информации

**Сколково:**
1. `skolkovo_mcd` — электричка, рекуперация, трение качения
2. `technopark` — атриум, эффект дымохода, вентиляция
3. `world_class_skolkovo` — ходьба/бег, число Фруда
4. `bolshoy_blvd_center` — пешеходные потоки, фундаментальная диаграмма
5. `hypercube` — тессеракт, 4D, фотовольтаика
6. `matrex` — пирамида, рекурсия, геометрическая прогрессия

## Решение по 42 лекциям

### ✅ Grounded (8 лекций, реально расширяют POI)

| Лекция | POI | Как расширяет |
|---|---|---|
| Парадокс Браесса (из «Как бороться с пробками») | `boulevard_game` (Иннополис) | Уже там как заявленный сюжет. Расширить L2/L3 конкретными цифрами Гасникова: 4000 авто, 65→80 мин |
| Оценка плотности леса по ближайшим деревьям (из «Вероятность вокруг нас») | `forest_water_column` (Иннополис) | Тот же лесной POI; рулетка к стволу + формула n≈1/(πr²). Активный замер на месте |
| Лотка-Вольтерра, хищник-жертва (из «НПТ в Чечне») | `forest_water_column` (Иннополис) | Лесной POI; добавить L3-сюжет про экологические циклы. Карточка с волнами «зайцы/волки» |
| Кузнечик и человек прыгают на ту же высоту (из «П-теорема») | `sports_scaling` (Иннополис) и/или `world_class_skolkovo` | Прямой синоним POI. Расширить L2 формулой `v² ∝ const`, не зависящей от размера |
| Геометрическая прогрессия 1/2+1/4+… (из «Избранные мат. этюды») | `matrex` (Сколково, пирамида с матрёшкой) | Прямая привязка: пирамида = рекурсия, матрёшки внутри = сходящийся ряд |
| Сеть Штейнера (из «Оптимизация в природе») | `skoltech` (Сколково, изопериметрия) | Тот же кампус — изопериметрия + Штейнер обе про оптимальные формы городского планирования |
| Точка Ферма-Торричелли с грузиками (из «Оптимизация в природе») | `skoltech` (Сколково) | Та же тема оптимизации формы кампуса/расположения |
| Комплексные числа = поворот на 90° (из «Конечно-разностные ур.») | `hypercube` (Сколково, тессеракт) | Гиперкуб = 4D-куб, проекция в 3D; комплексные числа = повороты в 2D. Мост между размерностями |

### ⚠️ Marginal (4 лекции, можно подвинуть, но не очевидно)

| Лекция | Возможный POI | Почему marginal |
|---|---|---|
| Алгоритм Дейкстры | `robot_path` или `boulevard_game` | Роботы используют RRT/A*, не Дейкстру; на бульваре POI уже перегружен |
| Принцип Ферма (закон Снеллиуса) | `skoltech` или новый POI у пруда | На маршрутах нет воды как явного объекта; нужно проверить пруды Сколково |
| Закон отражения через Ферма | — | Нет зеркальной поверхности на маршруте |
| Опыт Галилея | `tesla_marker` (Иннополис) | Tesla marker про индукцию, не падение. Башня Сююмбике — Казань, не на маршруте |

### ❌ Archive (30 лекций, нет приземления)

Чистые алгоритмы/теория игр/политика/абстрактные парадоксы — нет реального места на маршрутах Иннополис/Сколково, критический посетитель пройдёт мимо. Список:

- **CS-алгоритмы**: backprop, аукцион Викри, Карацуба, деление Ньютона, Штрассен, PCA, Netflix, k-means, HyperLogLog, power-of-two-choices, pooled testing
- **Чистая теория игр / соц.**: tit-for-tat, Хотеллинг, Вороной, Гейл-Шепли, Эрроу, антагонистическая игра, мудрость толпы, локальное большинство, игра «Жизнь»
- **Чистая вероятность**: Бертран, шахматный парадокс, встреча, Паскаль-Ферма, три карты, Симпсон, Монти-Холл
- **Чистая математика**: формат A4, 5 Платоновых тел, верёвка вокруг Земли, масштаб карты, coin rotation, Шварц-Фаньяно, диофанта (Гёдель), Тейлор, цепные дроби
- **Прочее**: SIR-эпидемия, эндорейное озеро (нет озера на маршруте), наклонные слои (нет обнажения), физика природы / Красная поляна (другой регион)
