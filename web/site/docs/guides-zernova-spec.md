# Фича «Гиды» + страница Елены Зерновой — спека

> Дизайн-документ. Новый тип контента «гид» на сайте Тропа (tropa.fmin.xyz),
> изолированный от научных сюжетов. Первый гид — Елена Зернова (Калининград).

## 1. Цель

Добавить на сайт раздел **«Гиды»** — каталог аттестованных экскурсоводов с авторскими
маршрутами. Каждый гид = карточка (фото + имя + регион) → персональная страница
(био справа, авторский маршрут как **интерактивная рисованная карта** в центре).

Это «агрегатор гидов» из видения Гасникова, в минимальной форме (Даниил: «не маркетплейс»):
маршрут гида + фото → обработка ИИ → публикация «маршрут гида, дополнен ИИ».

## 2. Почему отдельный тип, а не сюжет

`content/storylines/` заточен под научные сюжеты тропы Иннополиса: маркер готовности —
формулы KaTeX, tone of voice Гасникова (баффлинг → контринтуитивный поворот → число).
Маршрут гида (городская экскурсия по Светлогорску, 20 архитектурных точек) не содержит
ни формул, ни науки. Класть его в `storylines/` нельзя — ломает конвенцию «формула = vetted»
и тон. → Новый изолированный тип `guides/`.

## 3. Модель данных

`content/guides/<slug>.yaml` (новый источник). Схема:

```yaml
slug: zernova
name: Елена Зернова
region: Калининградская область
attested: true                      # аттестованный гид
portrait: assets/guides/zernova/portrait.jpg
# 3-4 факта доверия (вытащены из CV, НЕ сырое CV)
credentials:
  - Коренная калининградка в третьем поколении
  - Аттестована при Правительстве Калининградской области
  - Переводчик с немецкого (письменный и устный)
  - 6 лет практики, индивидуальные и групповые экскурсии
# продающий абзац (голос гида, блок «B» из присланного)
bio_md: |
  Уже 6 лет показываю этот край так, что его не просто смотрят — его проживают…
contacts:
  phone: "+7 906 210-60-28"
  # email/соцсети — отсутствуют, блок скрыт пока не появятся
# авторский маршрут → интерактивная карта
route:
  title: Пешеходный Светлогорск
  duration: 2,5–3 часа
  map_image: assets/guides/zernova/map.png   # фон-карта (Nano Banana 2)
  points:
    - n: 1
      title: ж/д вокзал Светлогорск-2 (старт)
      image: assets/guides/zernova/points/01.png
      x: 8                # позиция пина на карте, % от ширины
      y: 82               # % от высоты
    - n: 2
      title: Скульптура «Царевна-лягушка»
      image: assets/guides/zernova/points/02.png
      x: 16
      y: 74
    # … 20 точек
```

20 точек маршрута (из присланного, по порядку):
1. ж/д вокзал Светлогорск-2 (старт) · 2. Царевна-лягушка · 3. Дюнный дом ·
4. Дом казначея Карла Кюна · 5. Макет средневекового Кёнигсберга · 6. Ул. Ленина ·
7. Дом бургомистра Раушена · 8. Курхаус и знак Томасу Манну · 9. Серпантинный спуск ·
10. Вилла «Хильдегардт» · 11. Водонапорная башня с солнечными часами ·
12. Скульптура «Улитка» · 13. Лиственничный парк · 14. «Несущая воду» Брахерта ·
15. Центральная площадь, янтарное сердце Варвары · 16. Органный зал Макарова ·
17. «Домик гномика» · 18. Театр «Янтарь-холл» · 19. Парк «Времена года» ·
20. Лифтовая группа и клифы (Балтийское море).

## 4. Шаблоны (зеркалим storylines → storyline)

- `templates/guides.html` — сетка карточек (копия паттерна `storylines.html`:
  `.sl-grid` → `.gd-grid`, карточка с фото-фоном + serif-имя + регион подписью).
- `templates/guide.html` — детальная страница:
  - центр: H1 «Маршрут {name}, дополнен ИИ» → подзаголовок `{route.title} · {duration} · 20 точек`
    → **интерактивная карта** (см. §6).
  - правый sidebar: портрет → имя + «аттестованный гид» → `credentials` списком →
    `bio_md` → контакты (телефон текстом; соцсети/email — скрыты, пока пусты).
  - вёрстка: на десктопе 2 колонки (карта + sidebar), на мобиле sidebar уходит вниз.

## 5. Изменения в build.py

- `load_guides()` — читает `content/guides/*.yaml` → список гидов.
- В `build()`:
  - `guides/index.html` ← `guides.html` (сетка).
  - `guides/<slug>/index.html` ← `guide.html` (деталь) для каждого гида.
- Навигация (`templates/base.html`): ссылка `<a href="/guides/" class="nav-link">Гиды</a>`
  после «Бот».

## 6. Интерактивная карта (механика)

1. **Фон** — `route.map_image`: художественная карта-серпантин Светлогорска
   (вокзал → море), сгенерирована Nano Banana 2 в стиле сайта.
2. Поверх фона — **HTML-пины 1→20**, позиционируются абсолютно по `point.x`/`point.y` (%).
3. Клик/наведение на пин → карточка-поповер: `point.image` (миниатюра) + `point.title`.
4. Чистый CSS+JS, без внешних карт-библиотек (это рисованная схема, не гео-карта).
5. Пометка под картой: «Иллюстрации сгенерированы ИИ» (точки — не документальные фото).

## 7. Генерация ассетов (Nano Banana 2)

Стиль-промпт (общий для всех картинок, попадание в дизайн-систему):
> hand-drawn watercolor-and-ink sketch, muted earthy palette (sand #DBD4C7, sage #B8B5A6,
> olive #7C7F6A, charcoal #3E4038) on warm white, soft linework, literary/storybook mood,
> no photorealism, no text labels.

Список ассетов:
- 1× фон-карта маршрута Светлогорска (серпантин с морем, узнаваемые силуэты: курхаус,
  водонапорная башня, клифы).
- 20× миниатюры точек (по названию объекта).
- 1× портрет Елены — обработка её реального фото (из присланных 4 шт) под аватар.

## 8. Маппинг присланного контента

| Присланный блок | Куда |

| 4 фото | портрет (из фото с лучшим ракурсом) |

## 10. Форма саморегистрации гида (этап 1b) — ОТЛОЖЕНА

Решение Магомеда (20.06.2026): форму саморегистрации гида делаем **позже, одним serverless-бэкендом
вместе с site-ботом** (оба упираются в «у статики нет бэкенда» — не строим бэкенд дважды).
Архитектура цели: нативная форма → serverless-функция → заявка + Telegram-апрув Магомеду
(«Принять/Отклонить») → по «принять» пишет `content/guides/<slug>.yaml` через GitHub API → пересборка.
Первый гид (Елена) заводится **вручную** — форма ей не нужна.
Кнопка «стать гидом» на странице `/guides/` появится с бэкендом, сейчас её НЕТ.

## 11. Вне scope / открытые вопросы

- **Соцсети и email Елены** — отсутствуют в присланном. Блок иконок скрыт до появления ссылок.
- **Реальные фото точек** — не ждём, генерируем ИИ (решение Магомеда).
- Картинки точек — атмосферные ИИ-иллюстрации, НЕ документальные фото памятников.
- Стиль картинок: сайт нативно использует **векторный SVG line-art** (иконки сюжетов чистятся
  `scripts/add_icon.py`); карта Елены — растровая ИИ-иллюстрация (другой жанр, согласовано).
