# Контент POI

Сюжет хранится в YAML, бот разворачивает его в L1/L2/L3.

## Обязательные поля

`id`, `name`, `lat`, `lon`, `trigger_radius_m`, `topic`, `scenario_brief`, `level1_seed`, `level2_seed`, `physical_interactive`, `expert`, `sources`, `verified`.

## Уровни

| Уровень | Назначение | Требование |
|---|---|---|
| L1 | голос у точки | 80-150 слов, без формул |
| L2 | `/level2` | формула + численная оценка |
| L3 | `/level3` | байка, источники, связки |

## Качество

- есть реальный объект рядом;
- есть число с единицами;
- есть формула в L2;
- есть физический интерактив;
- понятно, почему это место, а не абстрактная лекция.
