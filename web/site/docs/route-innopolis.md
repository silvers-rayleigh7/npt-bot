# Маршрут Иннополиса

> ⚠️ **Это наш демо-концепт по кампусу, а не реальная тропа.** Реальная тропа
> Иннополиса — природная, по правому берегу Волги (4 км + 12 км петли),
> с другими станциями (Маятниковая роща, Услышь расстояние, Солнечные часы…).
> Фактический статус, реальные станции и команда: [`innopolis-real-status.md`](innopolis-real-status.md).

Файл маршрута: [`bot/config/poi.innopolis.yaml`](../bot/config/poi.innopolis.yaml).

## POI

1. `university_entrance` — GPS/ГЛОНАСС.
2. `atrium_library` — акустика атриума.
3. `robot_path` — SLAM и лидар.
4. `tesla_marker` — индукция Фарадея.
5. `boulevard_game` — Нэш-Вардроп и Браесс.
6. `sports_scaling` — ходьба, бег, число Фруда.
7. `forest_water_column` — вода в деревьях.
8. `data_center_view` — дата-центры, тепло, энтропия.

## Полевой чек

- пройти маршрут с live location;
- записать фактические `lat/lon`;
- подобрать место крепления;
- поправить радиусы;
- поставить `verified: true` только после проверки на месте.

Готовая POI = видимый объект, короткий L1, формула в L2, источник, эксперт, понятный физический интерактив.
