# Физические интерактивы

Правило: **один эффект, крупные детали, без хрупкой электроники, понятно за 20 секунд**.

## Материалы

Фанера, металл, оргстекло/поликарбонат, трос, шарики, магниты, гравировка, антивандальный крепёж.

## Первая очередь

| POI | Макет |
|---|---|
| `university_entrance` | плита-трилатератор |
| `boulevard_game` | доска Браесса с шариками |
| `tesla_marker` | катушка с магнитным поршнем |
| `sports_scaling` | маятниковая нога |

## Не делать

Сенсорные экраны, мелкие съёмные детали, длинные инструкции, зависимость смысла от питания, декор без опыта.
