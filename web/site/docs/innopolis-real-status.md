## Главное расхождение с нашим прототипом

Наш `tropa.fmin.xyz` собрал **свой** маршрут по кампусу (8 точек: Главный вход,
Атриум, Дорожка роботов, Точка Теслы, Развилка, Спортзона, Зелёная зона, ИТ-парк) —
это **наш демонстрационный концепт**, придуманный удалённо, он не совпадает с реальной тропой.

**Реальная тропа Иннополиса — это природная тропа** по правому берегу Волги
(окраина Введенской слободы, склоны оврагов, речная долина), а **не** маршрут
по зданиям кампуса. Петли: **основная 4 км + большая 12 км**. Прохождение основной
петли с остановками — 1,5–2 часа.

Это надо держать в голове: наши «Главный вход / Атриум / Дорожка роботов» к реальной
тропе отношения не имеют. Реальные станции — ниже.

## Вычитанный контент в репозитории (2026-06-01)

Команда Иннополиса (Ляйсан Сафиуллова) передала **два вычитанных дока** — описания
экспонатов и библиотеку сюжетов для QR. Тексты прошли первичную очистку от галлюцинаций
модели и вычитку. Мы инкорпорировали их в репозиторий как эталонный контент:

- **Сырьё (verbatim):** `content/sources/innopolis_team/exhibits.md`,
  `storylines_library.md` + сплиттер `split.py`.
- **40 сюжетов для QR** → `content/qr/storylines/ST-01…ST-40.md` (полный текст,
  уровни A/B/C) + манифест `content/storylines.vetted.yaml` (метаданные, теги, связи
  с банком идей `storylines.yaml` и с физическими экспонатами).
- **41 экспонат** (14 ✅ + Солнечная система + черновики) →
  `content/exhibits/<code>.md` + манифест `content/exhibits.yaml`.
- **Сайт:** каждый сюжет = веб-страница `/qr/ST-NN/` (то, что открывается по QR у точки);
  индекс `/qr/` сгруппирован по разделам. Сборка — `python3 build.py`.

Это контент «на голову выше» нашего brainstorm-банка из 147 идей: у каждого сюжета
есть выверенный текст, формула, открытый вопрос и привязка к месту.

## ИИ-гид: что уже работает

**Уже есть рабочий прототип Telegram-бота — `Nature_Gid_bot`.** Его собрал Гасников
через Abacus.ai DeepAgent (Pro). Что умеет:

- голосовой вопрос → распознавание → ответ в **двух уровнях** (школьник 9–11 кл. + студент-математик);
- стиль: Квант/Квантик/Химия и жизнь, упор на математику/физику/астрономию/биологию;
- режим «гид» (`/guide`, `/гид`), геолокация, история интересов пользователя.

**Проблема бота:** все кредиты списываются с аккаунта Гасникова и быстро кончаются —
нужна модель «аккаунт на пользователя», а не общий кошелёк.

Кроме бота, есть **рабочий скилл `outdoor-scipop-architect`** (зип в `scipop_tropa/`) —
пайплайн создания и верификации научных экспонатов через ИИ.

### Где сюда встраивается наш прототип

То, что Даниил предложил Тимуру (15.05) и что мы умеем делать удалённо:

1. **Озвучка сюжетов голосом учёного** (ElevenLabs/Salute) — берём готовые
   вычитанные тексты ✅-станций, озвучиваем.
2. **GPS-триггер через PWA** — наушник + телефон в кармане, рассказ запускается у точки.
   Наш `bot/` уже это делает (Live Location → Haversine + cooldown).
3. **Маршрут по естественному запросу** («5 км, физика+астрономия, с коляской, ~час»)
   — LLM-агент поверх коммивояжёра по существующим точкам.

Ограничение, которое мы уже зафиксировали Тимуру: офлайн работает только для
pre-generated mp3; живой диалог с ИИ требует сети (гибрид или покрытие на тропе
через регионального оператора).

## Что делать дальше (для нас)

- **Не выдавать наш кампусный маршрут за тропу Иннополиса.** Либо переключить демо на
  реальные ✅-станции (Маятниковая роща, Услышь расстояние, Солнечные часы — у них есть
  готовый вычитанный текст), либо явно пометить кампусный маршрут как «демо-концепт».
- Координаты реальных станций — в Яндекс-картах внутри экспозиционного дока
  (геопривязка у каждой ✅-станции). Точные lat/lon — после выгрузки геослоёв (запланировано).
- Самый быстрый осязаемый вклад: озвучить 2–3 готовые ✅-истории голосом учёного и
  отдать Тимуру как демо GPS-гида.
