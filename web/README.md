# Web-сайт НПТ

Статус: 📋 план, кода ещё нет. Делаем с нуля после Obsidian-базы.

## Идея (из встречи 10.06)
- Не лендинг, а **интерактивный сайт с ИИ**, меняющийся под человека (задаёт вопросы, адаптирует контент).
- **Wikipedia-модель для гидов:** экскурсовод (кейс Елены, Калининград) регистрируется, заполняет свою
  страницу/статьи/контакты; модератор верифицирует. Скил создаёт страницу по ответам гида (аналог Tilda
  под НПТ). Сертификация гидов Центром.
- **Подписочная модель (SaaS):** часть бесплатно (привлекать), часть платно (например, открывать контакты
  за подписку). Freemium. НЕ давать через сайт доступ к платному Claude Opus — всё на своих скилах.
- Ассоциация с доменом **Наука.РФ** через Миннауки.
- Скилы автонаполняют контент (ищут сюжеты, верифицируют, выкладывают) — «даже когда мы спим».
