## Сделано (на 21.06.2026)
- **Раздел «Гиды» + страница Елены Зерновой** (Калининград) — Wikipedia-модель в действии:
  интерактивная карта Светлогорска (20 точек), портрет, био, контакты. 🟢 в проде.
- **24 научных сюжета** в стиле Даниила (3-уровневая подача Кратко/Как работает/Глубже +
  KaTeX-формулы + сноски с источниками + Typst-схемы + гравюра-иконки):
  Калининград 10, Новосибирск/Академгородок 8, Якутия 6. 🟢 в проде.
- **Site-бот — текстовый ИИ-гид** (виджет «Чем помочь?» справа-внизу на всех страницах):
  читает текст открытого сюжета, отвечает кратко в тоне научного гида.
  Бэкенд — FastAPI-прокси, **GigaChat-2-Max → OpenRouter/DeepSeek fallback**,
  под systemd, наружу через nginx `/api/`. 🟢 в проде.
  Это «текстовый брат» голосового [`tg-bot/`](../tg-bot/) — тот же тон, но лаконично.

## Движок (репо Даниила)
`build.py` (Jinja + markdown) собирает статику из `content/` + `templates/`.
Сюжет = `content/storylines/<slug>.md`. Карта — Leaflet. Схемы — Typst+CeTZ (строго 0.13.1).
Site-бот — `site-bar/` (фронт `site/assets/site-bar/`, бэкенд `site-bar/server/`).
