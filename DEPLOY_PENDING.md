# Изменения к деплою (pending) — голос/текст/формулы + подбор голоса

Подготовлено локально, протестировано насколько возможно без сервера. Деплой — отдельным
заходом по SSH на сервер с ботом (VPS, `root@144.124.250.77`, systemd `claude-tg`).

## Что меняем

### 1. Текст ВСЕГДА + голос ВСЕГДА; формулы — только в тексте
- **CLAUDE.md** — новая логика: на каждый ответ уходят ДВА сообщения — голосовое (без формул)
  и текстовое (та же суть письменно). Формулы (математические/физические) пишутся ТОЛЬКО в
  текстовом сообщении и только когда уместны. Нет формулы → текст всё равно отправляется.
- Формулы — Юникод (`λ √ ⁴ ∝ π ·`), в моноширинном блоке, без LaTeX (Telegram его не рисует).

### 2. Персонализация
- **CLAUDE.md** — 3 уровня (новичок / любознательный / знаток) с явными сигналами определения
  (лексика, сложность вопроса, маркеры «по-простому»/«с формулой», углубление по ходу диалога)
  + правило зеркалить язык собеседника.

### 3. Голос «в духе Капицы» + механизм A/B
- **CLAUDE.md** — раздел «Манера речи в духе Сергея Капицы»: размеренный темп, пауза перед
  ключевым словом/термином, спокойная уверенность, уважительный регистр (манера, НЕ личность).
- **tools/voice_ab.py** — генерит один научный абзац голосами `filipp / zahar / ermil` (Yandex v1)
  на размеренной скорости → выбрать тембр на слух. Движок не меняем (надёжный v1).
- Юридика: клон реального голоса (Капицы и др.) для демо ПМЭФ НЕ используем — правовой и
  репутационный риск. Берём близкий синтез-голос + манеру.

### Новый MCP-инструмент
- **patches/apply_patches.py → patch_mcp_sendmsg** — добавляет в `mcp_server.py` инструмент
  `send_telegram_message(text)` (текст в активный чат). Идемпотентно. Нужен для пункта 1.

## Команды деплоя (выполнить на сервере)

```bash
cd /root/tropa-bot
git pull                                   # подтянуть CLAUDE.md, patches, tools
python3 patches/apply_patches.py           # наложит send_telegram_message + проверит AST
systemctl restart claude-tg
journalctl -u claude-tg -n 30 --no-pager   # убедиться, что поднялся без ошибок
```

### Подбор голоса (там же)
```bash
set -a; . ./.env; set +a
python3 tools/voice_ab.py                   # filipp/zahar/ermil на speed 1.0
# прослушать voice_samples/*.ogg, выбрать; затем зафиксировать в .env:
#   YANDEX_VOICE=<выбранный>
#   YANDEX_SPEED=<выбранный темп, напр. 1.0 — размереннее текущего 1.25>
systemctl restart claude-tg
```

## Проверка вживую (анти-самообман — без этого не считаем «работает»)

- [ ] Вопрос С формулой («почему небо голубое»/«период маятника») → пришли ДВА сообщения:
      голос БЕЗ формул + текст с формулой в ``` ```-блоке (`I ∝ 1/λ⁴`, `T = 2π√(L/g)`).
- [ ] Вопрос БЕЗ формулы («почему радуга цветная») → голос + текст (та же суть, без формулы).
- [ ] Голос не зачитывает символы; текст не дублирует весь голос слово-в-слово.
- [ ] Простой бытовой вопрос vs «строгий с терминами» → разный уровень ответа (персонализация).
- [ ] Выбранный голос/темп звучит размеренно, «по-профессорски».
