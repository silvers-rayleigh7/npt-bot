# НПТ — память и контекст проекта

> Это durable-контекст проекта НПТ, живёт В РЕПОЗИТОРИИ (не в Claude-памяти solyanka).
> Любая сессия Claude, открытая из этой папки, видит его. Обновляется по ходу работы.

## Что за проект
Центр научно-популярного туризма (НПТ), рук. **А. В. Гасников**. Продукты: научные тропы +
центры популяризации (музеи) + цифровые продукты. Магомед Акаев и Даниил Меркулов — сайт/бот.

## Монорепо (3 направления)
- `tg-bot/` — Telegram голосовой ИИ-гид (Claude + Yandex SpeechKit). В проде.
- `app/` — офлайн-приложение (предзагрузка + on-device TTS). План.
- `web/` — Wikipedia-сайт для гидов. План.
- `knowledge/` — **Obsidian-база сюжетов** (единый источник для всех трёх). Конвенция — `knowledge/90-meta/`.
- `docs/` — этот файл.

GitHub: `silvers-rayleigh7/npt-bot` (переименован из tropa-bot). Деплой бота: scp в `/root/tropa-bot` из `tg-bot/`.

## Решения по технологиям
- **Голос (онлайн-бот):** Yandex SpeechKit + выбор голоса (`/voice`). Пресет-голоса записи НЕ требуют.
- **Голос (офлайн-приложение):** **Piper** (лёгкий, CPU, быстрый — подходит слабому серверу 2ГБ/1CPU и on-device). НЕ XTTS (нужен GPU).
- **Свой/клонированный голос:** нужна чистая запись + обучение на арендованном GPU (дёшево, разово) для Piper, либо ElevenLabs API. Только с согласия владельца голоса.
- **Офлайн на тропе (костыль 17 июня):** бот заранее отдаёт «пакет» (предсгенерированные аудио-сюжеты + карта); связь упала — слушаешь скачанное; либо телефон гида как точка доступа.

## Как НПТ отделён от solyanka
Код и контент — в этом репо (npt-bot), коммиты идут СЮДА (не в solyanka). Память проекта — этот файл
(в репо). Эта сессия Claude технически запущена из воркстри solyanka, поэтому Claude-память сессии
пишется в стор solyanka. Чтобы НПТ был полностью самостоятелен — будущие сессии открывать из папки
этого репо (тогда и Claude-память будет НПТ).
