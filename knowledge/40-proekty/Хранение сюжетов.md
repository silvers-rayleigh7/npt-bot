# 🗄 Хранение сюжетов

Где и как хранить базу [[Сюжеты как единица контента|сюжетов]]. Решение встречи [[Встреча 10.06]].

## Решение
- Сейчас храним в **Git-репозитории** (контроль версий, markdown), не в простых папках Google Drive и не в Google Docs.
- Минус GitHub: визуально нельзя редактировать (скачать-править-залить).
- **Компромисс:** написать «переводчик»/синхронизатор Google Docs ↔ markdown на GitHub (правишь в Google Docs - отражается в markdown).
- Сначала всё сделать, потом дать доступ.
