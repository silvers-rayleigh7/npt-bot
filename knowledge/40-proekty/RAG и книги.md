# 📚 RAG и книги

«Джентльменский набор» источников для дообучения / RAG системы и [[Чат-бот гид|бота]].

## Книги
- **Ливанов** (учебники) - защищён (изд. «Просвещение»), договариваться через [[Александр Гасников]].
- **Перельман** - public domain.
- **Аржанов** - «главная книга по научно-популярному туризму» (география вековой давности).

## Лекции
- Лекции [[Александр Гасников|Гасникова]] с сайта «Физмат-прорыв» (десятки лекций по физике/биологии/географии). Ролики уже распарсили, есть несгенерированные несверенные сюжеты.

## Лицензии / правила
- Книги в электронном виде есть в ВК (PDF) - размещаем **ссылки**, не файлы (скилом искать и обновлять).
- Уровень контента - не круче журнала «Квант», но не попса.
