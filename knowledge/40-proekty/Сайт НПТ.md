# 🌐 Сайт НПТ

Интерактивный сайт с ИИ, меняющийся под человека (не лендинг). Текущая версия - MVP/лендинг с малым контентом. Берут под оперативный контроль [[Даниил Меркулов]] + [[Магомед Акаев]]. Вопрос «что показывать к 17-му» поставила [[Алина Ханова]].

## Концепция
- Ключевая единица - [[Сюжеты как единица контента|сюжеты]] плиткой. Можно интерактивно задавать дополнительные вопросы (как электронный учебник).
- Зеркало [[Чат-бот гид|Telegram-бота]] на сайте, режим проверки знаний для учителей.
- Набор скилов на платных подписках Claude + Open Cloud/Hermes автоматически наполняют сайт (ищут, верифицируют, выкладывают) - «даже когда мы спим».
- Wikipedia-модель: любой гид (как [[Елена (экскурсовод)]]) регистрируется, заполняет информацию/контакты/статьи; модератор верифицирует. Подписочная модель (SaaS).
- Сайт-«веточка» для гида: Google Form → скил создаёт страничку, скил-редактор, аналог Tilda под НПТ. Домен Наука.РФ через Миннауку, сертификация гидов Центром.

## Сначала
Нарисовать карту сайта, собрать контент.

## Сделано (июль 2026)
- **Гео-режим карты маршрута — целиком** (`tropa.fmin.xyz/innopolis/`): гео-чат «Рядом», карта центрируется на пользователе («📍 Я здесь»), пеший маршрут «от меня» к ближайшей точке через Valhalla. Спека — `docs/geo-mode-spec.md`, фазы 0–3 закрыты.
- **Методичка для учителя** (`/teacher/`): генерация урока + PDF. Веб-источники в опоре жёстко ограничены allowlist из 12 доверенных научпоп/образовательных доменов — см. [[Верификация контента]]. Убран экспорт «Текст (.txt)», оставлен PDF.
- Раздел «Гиды» с профилем [[Елена (экскурсовод)]] (Wikipedia-модель гидов).
