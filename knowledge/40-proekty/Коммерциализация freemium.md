# 💰 Коммерциализация (freemium)

Модель монетизации Центра. [[Александр Гасников]] предлагает прогнать «манифест» (п. 2.2) через скил системного / бизнес-аналитика, заложить через методику продуктологии.

## Принципы
- **Freemium:** часть бесплатно для всех (привлекать людей), часть платная/премиум.
- Скилы Центра не раскрываются клиентам; «не давать через свой сайт доступ к платному Claude Opus» - всё на своих закрытых скилах.
- Подписочная модель (SaaS) на [[Сайт НПТ]] - например, открывать контакты гидов за подписку.
- Сертификация гидов Центром (домен Наука.РФ) - уход от 25% комиссии TripAdvisor-подобных площадок (см. [[Елена (экскурсовод)]]).
