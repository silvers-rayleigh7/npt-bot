# 📍 Музей Мирового океана

Калининград. Директор - [[Светлана Геннадьевна]] («может президенту что-то направить»). Один корпус - 100 тысяч кв.м.

## Статус / дедлайн
- К концу года сдают музей; два этажа не доделаны - этажи о свойствах воды.
- [[Александр Гасников]] берёт контент по свойствам воды во многом на себя, направит [[Светлана Геннадьевна]] на этой неделе.

## Чем интересна (контент по свойствам воды)
- Максимальная плотность при +4 °C (озёра не промерзают до дна).
- Гидродинамические эффекты, формула Рэлея (?), гравитационные волны на воде.
- Соображения размерности (связь скорости волны с её длиной).
- Биология: почему вода важна для жизни (среда для клеток, уникальность Земли).
- См. сюжет [[Уникальные свойства воды]] и [[MOC - Вода]].
