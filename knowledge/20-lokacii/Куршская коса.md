# 📍 Куршская коса

Калининградская область. Им «очень понравилась идея научной тропы» (переписка [[Тимур]]). Несколько локаций-троп.

## Чем интересна
- Реальный кейс для бота: интегрировать в приложение Куршской косы интерактив, отвечающий на вопросы вокруг локаций - RAG-ом дообучаем ИИ, автономный режим без интернета, голосом ([[Чат-бот гид]]).
- Голос для бота взят у профессионального экскурсовода Куршской косы (с его разрешения) - см. [[Магомед Акаев]], Яндекс SpeechKit.
- Прочие точки: Шенецкая крепость (?), психбольница как достопримечательность (большой турпоток, нужна реставрация).
