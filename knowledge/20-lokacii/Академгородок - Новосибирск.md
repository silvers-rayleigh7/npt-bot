# 📍 Академгородок - Новосибирск

Маршрут «Золотая долина» с QR-кодами (вопросы-ответы и сюжеты). Локальная команда - [[Лейсан]], [[Константин Цыбиков]], [[Ольга Шевелева]] (знают Академгородок). «Академгородок - это десятка» по уровню.

## Дедлайн / цель
- **12 июня** - оформить предложения для презентации на Сибирском отделении РАН.
- Цель: чтобы Пармон (президент СО РАН) провёл по тропе президента, а [[Александр Гасников]] с членами Совета при президенте перехватывает рассказ.
- Связан с [[Визит президента (август)]] (СКИФ, Кольцово).

## Чем интересна
- Дорожки названы именами учёных (Лаврентьев, Канторович, Соболев, Беляев) - проходим жизненный путь учёного.
- Памятник академику Беляеву с лисами - готовая точка привязки арт-объекта.
- Академик Боровков - «сибирский вайб».
