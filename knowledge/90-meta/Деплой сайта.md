# Деплой сайта Тропа

Как выложить сайт на боевой сервер. **Пароль сервера здесь НЕ хранится** — этот файл в git (публичный репо), секрет держим локально.

## Куда

- **nginx-статика:** `root@144.124.250.77:/var/www/tropa/site` — плоские файлы (не git-clone).
- **backend (API урока/чата):** systemd на том же сервере, `/opt/tropa-bot/server`, uvicorn `127.0.0.1:8000`, наружу через nginx `/api/`.
- Доступ — **только по паролю** (ssh-ключи не подходят).

## Чем

Одна команда (читает креды из локального `~/.tropa-deploy.env`, вне git):

```bash
bash ~/Projects/tropa-bot/tools/deploy-site.sh          # только *.html
bash ~/Projects/tropa-bot/tools/deploy-site.sh --all    # + ассеты/css/js/pdf
```

Полный цикл при изменении шаблонов:

```bash
cd ~/Projects/tropa && PATH="$PWD/.tools:$PATH" python3 build.py
git checkout -- $(git status --short | grep '\.pdf$' | awk '{print $2}')   # убрать байтовый PDF-шум
bash ~/Projects/tropa-bot/tools/deploy-site.sh
```
