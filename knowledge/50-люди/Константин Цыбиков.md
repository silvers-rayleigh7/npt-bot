# 👤 Константин Цыбиков

Верификация контента, хранение сюжетов. Локальный знаток Новосибирска / [[Академгородок - Новосибирск]].

## Зона ответственности
- [[Верификация контента]] - ведёт направление, признано главным bottleneck'ом проекта.
- Хранение сюжетов ([[Хранение сюжетов]]): активный пользователь Obsidian, но граф связей «для личных заметок бесполезен».
- Скил SkillFeedReviewer - ранжирует приоритеты исправления ошибок.
- Один из локальной команды по [[Академгородок - Новосибирск]] (с [[Ольга Шевелева]]).
- Часть команды [[Тропа Иннополис]] (12 проработанных экспонатов): [[Лейсан]], Дарья, Константин.

## Верификация - детали
- Использует 5 разных аккаунтов в нейросетях: Claude и ChatGPT не видят ошибки друг друга.
- Нужен «консенсус» моделей + ранжирование ошибок по важности.
- Просит первокурсников помочь автоматизировать верификацию.
