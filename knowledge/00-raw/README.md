# 00-raw — сырьё (Layer 1 метода Карпати)

Сюда складываются **сырые источники как есть**, без обработки: транскрипты встреч, тексты статей,
выгрузки документов (спека тропы, Google-док сюжетов), материалы с сайта Даниила, лекции «Физмат-прорыва».

## Как это работает (RAW → компиляция → вики)
1. **RAW (здесь):** кладёшь сырой файл (`.md`/`.txt`).
2. **Компилятор = Claude Code:** по команде «обработай RAW и создай/обнови вики» — Claude читает
   сырьё, выделяет атомарные сюжеты/концепции, ставит связи `[[...]]`, дописывает в нужный MOC,
   **обновляет существующие заметки и отмечает противоречия** (не плодит дубли).
3. **Вики (`10-syuzhety/`, `30-karty/` …):** готовые связанные заметки.
4. **Линтер (`obsidian-keeper`):** показывает, какой RAW ещё не прокомпилирован (📥), сирот, пустышки, битые ссылки.

## Правила
- Файлы здесь — НЕ вики-заметки (линтер их не проверяет на сироты/frontmatter).
- Большие бинарники (docx/pdf/видео) сюда не коммитим — кладём текстовую выжимку или ссылку.
- После компиляции сырьё можно оставить (история источника) — линтер просто перестанет помечать его как «ждёт компиляции», когда появится одноимённая вики-заметка.
