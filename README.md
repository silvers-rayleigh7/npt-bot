# Бот НПТ — проект «Научно-популярный туризм»

Монорепозиторий проекта НПТ (Центр научно-популярного туризма, рук. А. В. Гасников).
Единый источник контента — «сюжеты» — проецируется на три продукта:

| Папка | Направление | Статус |
|-------|-------------|--------|
| [`tg-bot/`](tg-bot/) | **Telegram-бот** — голосовой ИИ-гид (Claude + Yandex SpeechKit). Онлайн-диалог. | ✅ работает в проде |
| [`app/`](app/) | **Приложение** — офлайн-режим на объекте (предзагрузка сюжетов + on-device TTS, GPS). | 📋 план |
| [`web/`](web/) | **Web-сайт** — интерактивный сайт + Wikipedia-модель для гидов (Наука.РФ). | 📋 план |

## Почему монорепо из трёх направлений
Telegram-бот не работает без интернета (ограничение платформы), офлайн на тропе умеет только
**Приложение** (кэш + локальный голос), а витрина и регистрация гидов — это **сайт**. Контент
(сюжеты) у всех общий — отдельный слой, переиспользуемый всеми тремя.

## Деплой (важно)
`tg-bot/` самодостаточен. На сервере бот живёт в `/root/tropa-bot` и наполняется через scp
**из `tg-bot/`** (корень сервера ← `tg-bot/`). Реструктуризация репозитория сервер не трогает —
живой бот продолжает работать. Детали — [`tg-bot/PROJECT.md`](tg-bot/PROJECT.md).
